# 控制迴圈
* 如何減少迴圈的錯誤:
	* 簡化，把能影響迴圈終止條件的因素減到最小，並把迴圈的內容當作是一個子程式，不必知道裡面的內容
	* 避免使用終止迴圈的變數，例如終止條件是一個變數var大於100，不要在迴圈結束後用var去判斷任何事情，會容易出現off-by-one的問題，解決方法是用另外一個變數去處理，例如用一個find的變數，當回圈內判斷條件達成，就將find變出設定成true並break，後面的程式只使用find變數去判斷，而不是用。
