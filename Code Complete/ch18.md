# 表格驅動法

表格驅動法是一種設計模式，相對於邏輯述句(if, case等)，他是在表格尋找資訊而不是用邏輯判斷。凡事能透過邏輯述句來選擇的事物皆可使用表格驅動法選擇。
例如文字分類，使用邏輯述句會有大量的判斷，像是要判斷a-z、A-Z、0-9、特殊符號。使用表格驅動法只要先把這些文字預先儲存至表格，在用查表的方式就可達到一樣的功能。

使用表格驅動法要先解決的兩個問題:
1. 怎樣從表格查詢項目
2. 表格內要儲存甚麼資料

## 怎樣從表格查詢項目
當資料數小時沒什麼問題，但是當資料數量龐大時要如何有效的查表，有三種方法可以查表:
* 直接存取
* 索引存取
* 階梯存取
